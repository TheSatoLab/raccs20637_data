# Online supplementary data for: *Virological characteristics of SARS-CoV-2-related coronaviruses dynamically circulating in Southeast Asia*

###  Wacharapluesadee *et al.* (2026) Cell. [10.1016/j.cell.2026.04.019](https://doi.org/10.1016/j.cell.2026.04.019)

Provided data: 

- [alignments](https://github.com/TheSatoLab/raccs20637_data/tree/main/alignments):

	- new PCR-amplified and NGS full genome sequences
	- full genome alignment of all sarbecoviruses


- [RBD and RdRp trees](https://github.com/TheSatoLab/raccs20637_data/tree/main/RBD-RdRp_trees):

	- Figure 1 RBD and RdRp alignments and trees

	
- [rdp5](https://github.com/TheSatoLab/raccs20637_data/tree/main/rdp5):

	- RDP5 recombination analysis and results
	- recombination-free alignment and tree


- [gard](https://github.com/TheSatoLab/raccs20637_data/tree/main/gard):

	- GARD recombination analysis input alignment and results
	- preliminary NRR alignments and trees
	- code for retrieving recombination breakpoints
	- sliding window recombination analysis
	
- [phylogeography](https://github.com/TheSatoLab/raccs20637_data/tree/main/phylogeo):

	- starting ML trees
	- code for creating BEAST xml
	- raw BEAST trees output
	- summarised BEAST log file
	- summarised BEAST trees
	- Spread.gl geojson summaries
	
- [metadata](https://github.com/TheSatoLab/raccs20637_data/tree/main/metadata):

	- sequence metadata
